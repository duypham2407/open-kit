## What's changed

- enforce hard runtime blocking for default code-search tools so agents must use OpenKit kit tools instead of default `grep`/`glob`
- enforce hard blocking of OS shell code-search/edit shortcuts (`grep`, `find -name`, `cat` on code, `head/tail`, `sed`, `awk`, `wc`, `ls`, `echo > code-file`) across quick, migration, and full workflows
- remove permissive/moderate bypass behavior for blocked categories in both OpenCode plugin enforcement and in-runtime bash guard hooks
- add and update enforcement tests to guarantee migration-mode and permissive-env overrides cannot bypass blocking
- include installation and runtime parity updates (global materialization, MCP bin/source parity, tooling shim correctness) and refreshed setup guidance for macOS/Ubuntu operators

## What's changed

- harden `/init-deep` as a direct-runtime repository-analysis command instead of a lane-selection surface
- teach `Master Orchestrator` and lane-routing docs to thin-dispatch direct-runtime commands without reinterpreting them as `/task`
- add install-bundle metadata and a regression check so packaged releases keep `/init-deep` routing, bundle, and operator-contract guarantees aligned
- enrich runtime-backed `AGENTS.md` refreshes with top-level repository discovery and project marker signals

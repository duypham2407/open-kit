## What's changed

- clarify the managed path model in runtime diagnostics so `openkit doctor` and `session-start` distinguish the global kit root, workspace state root, and project compatibility shim
- remove a hard-coded machine path from runtime-platform tests so cross-machine verification no longer fails on missing local repository paths
- lock the path-boundary guidance into operator docs, maintainer guidance, and governance tests so future changes cannot silently collapse global, workspace, and shim layers

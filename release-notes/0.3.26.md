## What's changed

- fix global OpenKit MCP startup so managed configs launch the MCP bridge from the installed npm package path instead of the mutable kit copy under `OPENCODE_HOME/kits/openkit`
- publish `.opencode/plugins/tool-enforcement.js` in the npm package so managed/global doctor and enforcement surfaces stay complete after install
- add global-install coverage for the managed MCP command shape and published plugin runtime surfaces

## Notes

- OpenClaw supervisor dialogue is advisory only: OpenKit remains the only authority for workflow state mutation, execution, approvals, evidence, task updates, and QA closure.
- Live hosted OpenClaw delivery is not required by this release; supervisor dialogue defaults to disabled/unconfigured and must degrade safely.
- Target-project application validation remains unavailable unless a target project defines app-native commands; OpenKit runtime checks must not be reported as target app proof.

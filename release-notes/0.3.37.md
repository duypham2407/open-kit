## What's changed

- make root `AGENTS.md` project-owned and commit-safe instead of a runtime-managed root shim
- turn `/init-deep` into a runtime-backed repository-guidance command with a checked-in handler that refreshes project-first `AGENTS.md` content while preserving the OpenKit workflow overlay
- add `tool.command-runner` as the deterministic in-session execution bridge for runtime-backed commands such as `/init-deep`
- preserve existing custom project guidance more safely during `AGENTS.md` refreshes and document the root-vs-compatibility guidance split across runtime and operator docs

## Notes

## What's changed

- preserve per-agent model overrides during `openkit upgrade`
- keep existing `agent-models.json` settings instead of resetting them during global install refresh
- add regression coverage for upgrade and install materialization so configured models survive future releases

## Validation

- `node --test tests/global/ensure-install.test.js`
- `node --test tests/cli/openkit-cli.test.js`

## Published package

- npm: `@duypham93/openkit@0.3.1`

## Notes

- this is a patch release on top of `0.3.0`
- existing users do not need to reconfigure agent models after upgrading to this version

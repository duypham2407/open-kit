## What's changed

- add reusable global agent model profiles through `openkit profiles --create|--edit|--list|--delete|--set-default`
- add session-scoped `/switch-profiles` so a running OpenKit session can choose an existing profile without mutating the global default or other sessions
- layer the global default profile into `openkit run`, preserve partial-profile fallback for omitted roles, and block deletion of default or active-session profiles
- document the new global CLI and in-session profile-switching surfaces across operator and validation runbooks

## Notes

- Target-project application build/lint/test validation remains unavailable unless a target project declares app-native commands.

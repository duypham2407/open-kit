## What's changed

- Establish the initial `0.2.8` baseline that was later hardened in `0.2.9`, `0.2.10`, `0.2.11`, and `0.2.12`.
- Provide the globally installed OpenKit operator flow built around:
  - `openkit run`
  - `openkit doctor`
  - `openkit upgrade`
  - `openkit uninstall`
- Materialize the managed global kit into the OpenCode home directory and launch OpenCode with the OpenKit profile.
- Ship the checked-in OpenKit agent, command, hook, and runtime surfaces needed for the global kit workflow.

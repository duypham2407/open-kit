## What's changed

- bundle `find-skills` directly into OpenKit so installed kits can proactively help users discover external skills when the current bundle is not enough
- bundle Vercel's React, Next.js, component-composition, and React Native skill guides into the default OpenKit install surface
- teach core implementation and solution-design roles to load the bundled Vercel skills when React, Next.js, React Native, Expo, or component-architecture work is in scope
- expand the maintainer/runtime skill maps so the bundled React and skill-discovery skills are part of the active AI-facing surface

- `npm run verify:all`

# Product Brief: Task Intake Dashboard

This file is part of the full-delivery golden-path example.

## Goal

Give operators one place to view incoming work items and their current processing status.

## Target Users

- Operations coordinator
- Delivery lead

## Problem Statement

Incoming tasks are scattered across multiple views, making triage slow and error-prone.

## High-Level Features
- [x] List incoming tasks with status and owner
- [x] Filter tasks by priority and queue
- [x] Surface blocked tasks quickly

## Out of Scope

- Automated assignment logic

# Archived Superpowers Design Material

This directory contains historical design and implementation-planning artifacts from the period when OpenKit was still borrowing structure and terminology directly from the older "superpowers" reference lineage.

Use these files only when you need maintainer background, historical rationale, or design archaeology.

They are not operator docs.
They are not the current live runtime contract.
They are not the authoritative source for shipped behavior.

For current live behavior, prefer:

- `README.md`
- `AGENTS.md`
- `context/core/workflow.md`
- `context/core/project-config.md`
- `docs/operations/README.md`

If these archived docs conflict with repository state, trust the repository state.
